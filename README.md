## Project Description
This project is a static fashion retail website developed using HTML and CSS. 
The purpose of this project is to demonstrate Git version control operations including branching, merging, conflict resolution, and GitHub integration.

## Technologies Used
- HTML
- CSS
- Git
- GitHub

## Git Operations Performed
- Initialized local Git repository using git init
- Created multiple branches (feature, test, bugfix, experiment)
- Performed more than 10 commits with proper messages
- Merged branches into main branch
- Demonstrated merge conflict and resolved it
- Connected local repository to GitHub using git remote
- Pushed all branches to GitHub
